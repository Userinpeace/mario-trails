# Mario Running Cursor

## 🚀 Overview

Mario Running Cursor is a fun and interactive web project designed as a simple yet feature-rich landing page. It combines engaging animations with smooth interactions, making it visually appealing and user-friendly. Mario dynamically follows the user's cursor across the screen, creating an immersive experience. Additionally, a button allows users to navigate to a new page seamlessly. The continuously scrolling background adds to the visual depth, making the interface lively and engaging.

## 🎮 Features

- 🏃‍♂️ Mario follows the cursor smoothly across the screen.
- 🎨 Stylish and animated background for an engaging experience.
- 🎭 Customizable CSS variables for easy theme adjustments.
- 🖱️ Click button to navigate to a new page.
- ⚡ Debounced mouse movement for better performance.

## 🛠️ Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express

## 🚀 Getting Started

### Prerequisites

- Node.js installed on your system

### Installation

1. **Clone the repository**

   ```sh
   git clone https://github.com/your-username/mario-cursor.git
   cd mario-cursor
   ```

2. **Install dependencies**

   ```sh
   npm install
   ```

3. **Run the server**

   ```sh
   node server.js
   ```

4. **Open in browser**

   - Go to `http://localhost:3000/`

## 🎨 Customization

You can modify the `styles.css` file to adjust colors, background, or Mario's speed.
